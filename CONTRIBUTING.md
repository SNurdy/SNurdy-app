# 🤝 Contributing Guidelines

감사합니다! 이 프로젝트에 기여하고자 해주셔서 환영합니다. 아래 지침을 따라주세요.

## 📌 기여 방법

1. 이슈 확인 또는 생성
2. `fork` 후 새로운 브랜치 생성 (`feature/my-feature`)
3. 로컬에서 수정 후 커밋
4. `pull request` 생성 및 리뷰 요청
